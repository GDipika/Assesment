Directives in Angular

ng-container

A special element that can hold structural directives without adding new elements to the DOM.

ng-content

What ng-content helps us do is get that HTML-like functionality into our components, making it more flexible to use components and pass data from one component to another. All you have to do is go into the component you want to be projected into and specify it like this to Angular.

ng-template

ng-template is an Angular element that is used for rendering HTML in a template. However, it is not rendered directly on DOM. If you include an ng-template tag to a template, the tag and the content inside it will be replaced by comment upon render.
